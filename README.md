# 🚀 CodeWeaver AI

An AI-powered web application generator that converts natural language ideas into structured web app designs, including features, architecture, and sample code.

---

## 🌟 Features

* 🧠 AI Agent (Rule-based reasoning)
* ⚙️ Automatic feature extraction
* 🏗️ System architecture generation
* 💻 Sample frontend & backend code generation
* ☁️ MongoDB cloud integration
* 🌐 Full-stack (Frontend + Backend)

---

## 🧠 How It Works

The system follows an **AI Agent pipeline**:

1. **Analysis Layer**

   * Understands user prompt
   * Extracts key features

2. **Planning Layer**

   * Designs architecture
   * Chooses tech stack

3. **Generation Layer**

   * Produces frontend & backend code
   * Suggests database schema

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** FastAPI
* **Database:** MongoDB Atlas
* **Deployment:** Render + GitHub Pages

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR_USERNAME/code-weaver-ai.git
cd code-weaver-ai
```

### 2️⃣ Create virtual environment

```
python3 -m venv venv
source venv/bin/activate
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Add environment variables

Create `.env` file:

```
MONGO_URI=your_mongodb_connection_string
```

### 5️⃣ Run backend

```
uvicorn main:app --reload
```

### 6️⃣ Open frontend

Open `index.html` in browser
